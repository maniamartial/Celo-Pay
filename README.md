# Ecommerce with Crypto Payment

```
Simple application which allows cutomer to buy products and pay using celo crypto


```

# Technologies

```
Frontend-React
Backend - Smart Contract, Solidity


```

# Home
```
Its the market place, where we have many products to be sold. They are stored in PSF whih is secure blockchain database.
You can choose any product, but if you want to buy, you get redirected to login page.
If you have logged in, you can easily click on the button "Purchase", you will be redirected to a form which you will enter a message,  and complete transaction. 
If you havent loggedIn and you dont want to create an account, yu will enter in your celo key pay in the textfield "from...". 
Then enter the id to whom youare sending the money,wh is the owner of the product. Complete the transaction and youyr item will be delivered. 

```
# Login
```

Use your key wallet and easily login. If you dont have, then you have to choose "Dont have an account"

```

# Signup
```
This will redirect you to celo pay where you will create a account and use the key for accessing the website.
After registering, you will get back to login page and login to the site.

```

# Importnat extension for the application to work
```
1. Install the [CeloExtensionWallet](https://chrome.google.com/webstore/detail/celoextensionwallet/kkilomkmpmkbdnfelcpgckmpcaemjcdh?hl=en) from the google chrome store.
2. Create a wallet.
3. Go to [https://celo.org/developers/faucet](https://celo.org/developers/faucet) and get tokens for the alfajores testnet.
4. Switch to the alfajores testnet in the CeloExtensionWallet.
```
